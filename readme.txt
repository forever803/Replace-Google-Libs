=== Replace Google Libs ===

Contributors: xzhang
Tags: USTC Libs, 360 Open Libs, Google Libs, Google Open Libs
Stable tag: 1.0

Use USTC Open Libs Service to replace Google's.

== Description ==

[Plugin homepage](https://github.com/forever803/Replace-Google-Libs) | [Plugin author](https://github.com/forever803)

There are some problem with Google in china, the Google Libs make the website too slow to open, so we can use USTC Libs replace the Google's.

[Project GitHub](https://github.com/forever803/Replace-Google-Libs).

== Installation ==

1. Upload `replace-google-libs` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
