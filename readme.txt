Use Qihoo 360 Open Libs Service to replace Google's.

== Description ==

[Plugin homepage](http://www.soulteary.com/2014/06/15/replace-google-libs.html) | [Plugin author](http://www.soulteary.com/)

There are some problem with Google in china, the Google Libs make the website too slow to open, so we can use Qihoo 360's Libs replace the Google's.

[Project GitHub](https://github.com/soulteary/Replace-Google-Libs).

== Installation ==

1. Upload `replace-google-libs` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
